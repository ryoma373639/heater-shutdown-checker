# Assets Directory

このディレクトリは静的リソースを格納します。

## 構造

```
assets/
├── css/         # スタイルシート
├── js/          # JavaScript ファイル
└── icons/       # PWA用アイコン画像
```

## 使用方法

### CSS

カスタムCSSファイルを`css/`ディレクトリに配置します。

### JavaScript

アプリケーションロジックを`js/`ディレクトリに配置します。

### Icons

PWA manifest用のアイコン画像を`icons/`ディレクトリに配置します。
