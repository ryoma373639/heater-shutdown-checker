# ユーザーガイド

灯油ストーブ消火確認アプリの詳しい使い方を説明します。

## 📖 基本的な使い方

### 社員の追加

1. 画面上部の「社員の追加」セクションを見つけます
2. テキストボックスに社員名を入力します
   - 例: 「山田太郎」
3. 「追加」ボタンをクリックまたはタップします
4. リストに社員が追加されます

**ヒント:**
- Enterキーを押しても追加できます
- 同じ名前でも追加できます（区別されます）

### 工場タブの使い方

アプリは4つの工場に対応しています：

- **第一工場** - 1つ目のタブ
- **第二工場** - 2つ目のタブ
- **第三工場** - 3つ目のタブ
- **第四工場** - 4つ目のタブ

各工場で独立して社員リストと確認状況を管理できます。

**切り替え方法:**
1. 画面上部のタブをクリック
2. タブの色が変わり、その工場のデータが表示されます

### 消火確認のチェック

1. 該当する工場のタブを選択
2. 消火確認が完了した社員のチェックボックスをオンにします
3. チェックすると：
   - 背景が緑色になります
   - 社員名に取り消し線が入ります
   - 確認済み人数が増えます

**進捗表示:**
- タイトルに「消火確認リスト (3/10)」のように表示されます
  - 3: 確認済み人数
  - 10: 総社員数
- 「未確認: 7名」と未確認人数も表示されます

### 最終確認と完了

退社時に必ず実行してください：

1. 「最終確認・完了」ボタンをクリック
2. **未確認者がいる場合:**
   - 警告ダイアログが表示されます
   - 未確認者のリストが赤色で表示されます
   - 「確認しました」ボタンで閉じます
3. **全員確認済みの場合:**
   - 完了メッセージが表示されます
   - 🎉 お祝いのアイコンが表示されます
   - 「閉じる」ボタンで閉じます

### 社員の削除

不要になった社員を削除できます：

1. 削除したい社員の行にある「削除」ボタンをクリック
2. すぐに削除されます（確認なし）

**注意:** 削除すると元に戻せません！

### 社員の再使用

番号付きで社員を追加できます（日替わり作業者などに便利）：

1. 「再使用」セクションを見つけます
2. **番号**: 作業者番号を入力（例: 「101」）
3. **名前**: 自分の名前を入力（例: 「田中」）
4. 「再使用」ボタンをクリック
5. 「101 田中」として追加されます

**用途例:**
- 日替わり担当者の管理
- 外部作業者の記録
- 部署番号付きの管理

## 🔄 データリセットのタイミング

### 自動リセット

**毎日AM3時に自動的にリセット**されます：

- すべてのチェックボックスがオフになります
- 社員リストは保持されます
- 全工場で同時にリセットされます

**なぜAM3時？**
- 深夜帯で作業者が少ない時間
- 前日分と当日分を明確に分離

### 手動でリセットしたい場合

現在、手動リセット機能はありません。
必要な場合は、各社員のチェックを手動で外してください。

## 💾 データの保存

### 自動保存

すべての操作は**自動的に保存**されます：

- 社員を追加した瞬間
- チェックボックスをオン/オフした瞬間
- 社員を削除した瞬間

**保存場所:** 端末内のlocalStorage（外部送信なし）

### データの永続性

- アプリを閉じても保存されます
- ブラウザを閉じても保存されます
- 端末を再起動しても保存されます

**注意:**
- ブラウザのデータをクリアすると消えます
- 別の端末では同期されません

## 📱 スマートフォンでの使い方

### ホーム画面に追加すると

- アプリアイコンがホーム画面に表示されます
- 通常のアプリのように起動できます
- オフラインでも使えます

### 横画面での使用

- 自動的にレイアウトが調整されます
- タブは横並びで表示されます
- より多くの社員を一度に表示できます

### タッチ操作

- **タップ**: ボタンやチェックボックスを操作
- **スワイプ**: リストをスクロール
- **長押し**: （現在未対応）

## 🎯 運用のベストプラクティス

### 朝の運用

1. アプリを起動
2. 前日AM3時に自動リセットされていることを確認
3. 必要に応じて新しい社員を追加

### 日中の運用

1. 各工場の担当者が自分の工場タブを開く
2. 消火確認が完了したらチェック
3. 定期的に未確認人数を確認

### 退社時の運用

1. 各工場で「最終確認・完了」を実行
2. 未確認者がいれば連絡して確認
3. 全員確認済みになったら退社

### 複数端末での運用

各端末で独立して動作します：

- **第一工場の端末**: 第一工場タブのみ使用
- **第二工場の端末**: 第二工場タブのみ使用
- **管理者の端末**: 全工場のタブを確認

## ❓ よくある質問

**Q: チェックを間違えて外してしまった**
A: もう一度クリックすれば元に戻ります。

**Q: 社員を削除してしまった**
A: 削除は取り消せません。再度追加してください。

**Q: 昨日のデータを見たい**
A: 現在、履歴機能はありません。AM3時にリセットされます。

**Q: 別の端末でも同じデータを見たい**
A: 現在、同期機能はありません。各端末で独立しています。

**Q: データをバックアップしたい**
A: 現在、バックアップ機能はありません。

**Q: 工場の数を増やしたい**
A: 現在、4工場固定です。カスタマイズは開発者向けドキュメントを参照。

**Q: 社員名を変更したい**
A: 削除して再追加してください（チェック状態は失われます）。

**Q: オフラインで使える？**
A: はい、初回アクセス後は完全オフラインで使えます。

**Q: データは安全？**
A: はい、すべて端末内に保存され、外部送信はありません。

## 🔧 トラブルシューティング

### チェックボックスが反応しない

1. ブラウザを再読み込み（F5またはCommand+R）
2. アプリを再起動

### データが消えた

1. ブラウザのデータをクリアしていないか確認
2. 別のブラウザで開いていないか確認
3. プライベートモードで使用していないか確認

### 自動リセットが動作しない

1. 端末の時計が正しいか確認
2. アプリを一度再起動
3. AM3時以降に初めてアプリを開いたときにリセットされます

### 表示が崩れる

1. ブラウザを最新版に更新
2. ページを再読み込み
3. キャッシュをクリアして再度アクセス
