# インストール手順書

灯油ストーブ消火確認アプリのインストール方法を説明します。

## 📱 スマートフォン向けインストール

### iPhone / iPad (iOS)

1. **Safariでアプリを開く**
   - https://ryoma373639.github.io/heater-shutdown-checker/ にアクセス

2. **共有メニューを開く**
   - 画面下部中央の「共有」ボタン（□↑）をタップ

3. **ホーム画面に追加**
   - メニューから「ホーム画面に追加」を選択
   - アプリ名を確認（変更も可能）
   - 右上の「追加」をタップ

4. **完了！**
   - ホーム画面にアイコンが表示されます
   - タップすると通常のアプリのように起動します

**トラブルシューティング（iOS）:**
- 「ホーム画面に追加」が表示されない → Safariを使用していることを確認
- アイコンが表示されない → ホーム画面を右にスワイプして探す

### Android スマートフォン

1. **Chromeでアプリを開く**
   - https://ryoma373639.github.io/heater-shutdown-checker/ にアクセス

2. **メニューを開く**
   - 画面右上の縦三点リーダー（⋮）をタップ

3. **ホーム画面に追加**
   - 「ホーム画面に追加」または「インストール」を選択
   - 確認ダイアログで「インストール」または「追加」をタップ

4. **完了！**
   - ホーム画面にアイコンが表示されます
   - アプリドロワーからも起動できます

**トラブルシューティング（Android）:**
- 「ホーム画面に追加」が表示されない → Chromeを最新版に更新
- インストールできない → ストレージ容量を確認

## 💻 パソコン向けインストール

### 方法1: オンラインで使用（最も簡単）

1. ブラウザで https://ryoma373639.github.io/heater-shutdown-checker/ を開く
2. ブックマークに追加（推奨）

### 方法2: ZIPファイルをダウンロード

1. **ダウンロード**
   - [GitHub Releases](https://github.com/ryoma373639/heater-shutdown-checker/releases)にアクセス
   - 最新版の `heater-shutdown-checker-vX.X.X.zip` をダウンロード

2. **解凍**
   - **Windows**: ZIPファイルを右クリック → 「すべて展開」
   - **macOS**: ZIPファイルをダブルクリック

3. **起動**
   - 解凍したフォルダ内の `index.html` をダブルクリック
   - ブラウザで開きます

4. **ショートカット作成（オプション）**
   - **Windows**: `index.html`を右クリック → 「ショートカットの作成」→ デスクトップに移動
   - **macOS**: `index.html`をDockにドラッグ

### 方法3: 開発者向け（ローカルサーバー）

```bash
# リポジトリをクローン
git clone https://github.com/ryoma373639/heater-shutdown-checker.git
cd heater-shutdown-checker

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev
```

ブラウザで http://localhost:3000 にアクセス。

## 🌐 オフライン利用について

### 初回アクセス後

1. アプリに一度アクセスすると、自動的にキャッシュされます
2. 以降はインターネット接続なしで利用可能
3. データはすべて端末内に保存されます

### キャッシュのクリア

オフライン機能をリセットしたい場合：

**スマートフォン:**
- ブラウザの設定 → サイト設定 → ストレージをクリア

**パソコン:**
- ブラウザのDevTools (F12) → Application → Storage → Clear site data

## 🔄 アップデート方法

### スマートフォンアプリ

アプリを起動すると自動的に最新版をチェックします。
手動で更新する場合：
1. アプリを閉じる
2. ブラウザで元のURL（https://...）にアクセス
3. 再度ホーム画面に追加

### ZIPファイル版

1. 新しいバージョンのZIPをダウンロード
2. 古いファイルを削除
3. 新しいファイルを解凍

**注意**: データは引き継がれません（localStorageに保存されているため）

## ⚙️ システム要件

### スマートフォン
- **iOS**: iOS 11.3以降、Safari
- **Android**: Android 5.0以降、Chrome 45以降

### パソコン
- **ブラウザ**: Chrome、Edge、Firefox、Safariの最新版
- **OS**: Windows 10以降、macOS 10.12以降、Linux

### ストレージ
- 必要容量: 約1MB
- データ保存: 端末のlocalStorage（通常5-10MB利用可能）

## 🛡️ セキュリティとプライバシー

- ✅ すべてのデータは端末内に保存
- ✅ 外部サーバーへのデータ送信なし
- ✅ 個人情報の収集なし
- ✅ オフラインで完全動作

## ❓ よくある質問

**Q: インストール容量はどのくらい？**
A: 約1MBです。非常に軽量です。

**Q: インターネット接続は必要？**
A: 初回アクセス時のみ必要です。以降はオフラインで利用できます。

**Q: データはどこに保存される？**
A: 端末内のlocalStorageに保存されます。外部には送信されません。

**Q: 複数の端末で同期できる？**
A: 現在は各端末で独立して動作します。同期機能はありません。

**Q: アンインストール方法は？**
A: スマートフォンの場合、通常のアプリと同様にアイコンを長押しして削除します。
